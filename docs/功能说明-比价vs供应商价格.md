# 功能说明：比价 vs 供应商价格

## 两个容易混淆的功能

系统提供了两个查看价格的功能，它们的用途不同：

### 📊 比价功能

**用途**：比较**不同平台**的价格

**适用场景**：
- 比较药师帮、京东、天猫等不同平台的价格
- 找出哪个平台最便宜
- 跨平台采购决策

**访问方式**：
- 导航栏 → 比价
- 或访问：http://localhost:5001/compare

**⚠️ 当前限制**：
- 目前系统仅采集了药师帮平台的数据
- **暂不支持跨平台比价**
- 如果搜索结果为空，说明该药品只有一个平台的数据

### 🔍 供应商价格查询

**用途**：查看**同一平台内不同供应商**的价格

**适用场景**：
- 查看药师帮平台上所有供应商的价格
- 找出最便宜的供应商
- 对比不同供应商的价格差异

**访问方式**：
1. 导航栏 → 搜索
2. 搜索药品名称
3. 点击药品进入详情页
4. 查看所有供应商价格列表

## 详细对比

| 对比项 | 比价功能 | 供应商价格查询 |
|-------|---------|---------------|
| **比较对象** | 不同平台 | 同一平台的不同供应商 |
| **数据来源** | 药师帮、京东、天猫等 | 药师帮平台 |
| **价格数量** | 每个平台1个价格 | 每个供应商1个价格 |
| **适用场景** | 跨平台采购决策 | 选择最优供应商 |
| **当前状态** | 仅药师帮数据 | 完整可用 |

## 常见问题

### Q1: 为什么比价功能搜索不到结果？

**答**：因为比价功能用于跨平台比较，而目前系统只有药师帮一个平台的数据，无法进行比价。

**解决**：使用搜索功能查看供应商价格。

### Q2: 如何查看某个药品的所有供应商价格？

**答**：
1. 访问搜索页面
2. 搜索药品名称
3. 点击药品进入详情页
4. 查看供应商价格列表

### Q3: 比价功能什么时候可以用？

**答**：当系统采集了多个平台的数据后，比价功能就可以使用了。目前可以扩展采集：
- 京东医药
- 天猫医药
- 1药网
- 等其他平台

### Q4: 供应商价格和比价有什么区别？

**答**：
- **供应商价格**：同一平台（药师帮）内不同供应商的价格
- **比价**：不同平台（药师帮 vs 京东 vs 天猫）的价格

### Q5: 我应该用哪个功能？

**答**：
- 如果你想在药师帮上找最便宜的供应商 → 使用搜索功能
- 如果你想比较不同平台的价格 → 使用比价功能（未来）
